## 📌 Overview

Chaaya is an AI-powered digital mourning application designed to help people cope with the loss of loved ones by creating a virtual emotional presence of the departed.

The platform uses AI, conversational interfaces, and memory-based inputs to simulate comforting interactions and support users in their grief healing journey.

------------------------------------------------------------

## 🎯 Problem Statement

Losing someone close creates a deep emotional void, and many people struggle with:

- Unresolved grief and emotional distress  
- Lack of closure  
- Absence of emotional support in private moments  
- Difficulty in expressing memories and feelings  

------------------------------------------------------------

## 💡 Our Solution

Chaaya provides a safe, private and emotionally supportive space where users can:

- Interact with an AI-based memory persona of their loved one  
- Preserve memories, conversations, and emotions  
- Receive comforting responses during vulnerable moments  
- Gradually move through stages of healing  

------------------------------------------------------------

## ✨ Key Features

- AI Chat Interface for emotional interaction

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/your-username/chaaya.git  
cd chaaya

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python main.py

4. Open in browser

http://localhost:3000

------------------------------------------------------------

## 🔗 API Endpoints

GET    /api/messages           → Retrieve chat history  
POST   /api/messages           → Send message to AI  
POST   /api/create-persona     → Create memory persona
